# MEPHI Session Project 2026

## Network Configuration Note

В процессе выполнения задания я столкнулся с проблемой при настройке сети в VirtualBox.

При попытке использовать Host-Only Adapter нужный сетевой интерфейс отсутствовал в списке доступных подключений, несмотря на его пересоздание и переустановку VirtualBox. Из-за этого не удалось корректно настроить сеть по заданному сценарию.

В связи с этим была использована сеть NAT, которая позволила обеспечить доступ к интернету и корректную работу всех необходимых сервисов (dnf, nginx, tcpdump).

Проверка веб-сервера выполнена через localhost, что подтверждает его работоспособность.

Остальные требования задания выполнены в полном объёме.
